Marks: Text Highlighter는 사용자가 모든 웹페이지에서 텍스트를 강조 표시하고 관리할 수 있게 해주는 강력한 개인정보 보호 중심 Firefox 확장 프로그램입니다. 생산성을 높이고, 읽기 워크플로를 개선하며, 중요한 내용을 다시 확인하는 과정을 매끄럽게 만들어 줍니다. 모든 데이터는 사용자가 통제할 수 있는 방식으로 설계되었습니다.

🔍 주요 기능
✅ 다중 색상 텍스트 하이라이트
어떤 웹사이트에서든 텍스트를 선택하고 원하는 색상으로 하이라이트할 수 있습니다. 이 확장 프로그램은 노란색, 초록색, 파란색, 분홍색, 주황색의 5가지 뚜렷한 하이라이트 색상을 제공하여 콘텐츠를 시각적으로 정리하고 분류할 수 있습니다.

⚡️ 컨텍스트 메뉴 또는 단축키로 빠른 하이라이트
선택한 텍스트를 다음 방법으로 즉시 하이라이트할 수 있습니다:
- 마우스 오른쪽 클릭 컨텍스트 메뉴
- 사용자 지정 가능한 키보드 단축키 (Ctrl+Shift+1 ~ Ctrl+Shift+4 등)

이 동작은 여러 탭과 세션에서도 작동합니다.

🗂 페이지별 하이라이트 관리
모든 하이라이트는 해당 URL과 함께 저장됩니다. 나중에 하이라이트한 페이지를 다시 열면 모든 하이라이트가 자동으로 복원됩니다. 다음과 같은 작업을 할 수 있습니다:

- 하이라이트한 모든 페이지 목록 보기
- 개별 하이라이트 또는 전체 그룹 검토, 수정, 삭제
- 페이지 제목, 마지막 업데이트 시각 같은 메타데이터 확인

🧭 내비게이션용 미니맵 보기
페이지 오른쪽 가장자리에 내장된 "미니맵"이 표시되어 모든 하이라이트의 마커를 보여줍니다. 이를 통해 다음이 가능합니다:

- 긴 페이지에서 하이라이트된 구간을 즉시 찾기
- 마커를 클릭해 해당 하이라이트로 바로 이동
- 현재 화면에 보이는 하이라이트 확인

🌐 다국어 지원
여러 언어를 지원하며, UI는 브라우저 언어 설정에 따라 자동으로 적용됩니다.

🧩 팝업 대시보드
확장 프로그램 아이콘을 클릭하면 다음에 빠르게 접근할 수 있는 팝업이 열립니다:

- 현재 페이지의 하이라이트된 텍스트
- 미니맵 토글
- 모든 하이라이트 삭제 또는 데이터 내보내기 버튼
- 전체 하이라이트를 관리하는 전체 페이지 관리자 이동

💾 로컬 우선 저장소 & 개인정보 보호
기본적으로 하이라이트, 색상, 페이지 메타데이터를 포함한 모든 데이터는 storage.local API를 사용해 브라우저 로컬에 저장됩니다. 브라우저 Sync를 활성화하면 데이터가 브라우저 계정을 통해 기기 간 동기화될 수 있습니다. Marks는 자체 백엔드 서버를 운영하지 않습니다.

💯 무료 & 오픈 소스
이 확장 프로그램은 완전 무료이며 MIT 라이선스 기반의 오픈 소스입니다. 프리미엄 기능 없음. 사용 추적 없음. 데이터 수집 없음.

🙌 왜 Marks를 설치해야 할까요?
- 핵심 콘텐츠를 즉시 표시해 읽기 생산성 향상
- 색상으로 하이라이트를 정리해 더 스마트한 학습
- 시각적 노트를 유지한 채 유용한 페이지 다시 확인
- 직관적인 미니맵으로 긴 페이지에서도 위치를 잃지 않음
- 로컬 우선 저장과 선택적 브라우저 Sync로 데이터 통제 가능
- 빠른 단축키와 컨텍스트 메뉴로 시간 절약
- 이 모든 기능을 평생 무료로 제공

🔐 신뢰 & 개인정보 보호
Marks는 개인정보 보호를 최우선 가치로 설계되었습니다:

- 🚫 브라우저 Sync 를 제외한 서버 통신 없음
- 🚫 분석(애널리틱스) 없음
- 🚫 로그인/계정 필요 없음

✅ 기본적으로 로컬에서 동작

이 확장 프로그램은 다음 사용자에게 이상적입니다:

- 연구자
- 학생
- 기사나 문서를 자주 읽는 전문가
- 웹 전반에서 핵심 정보를 보존하고 정리하고 싶은 모든 사용자

지금 Marks를 설치하고 빠르고, 프라이빗하며, 아름답게 정리된 방식으로 웹을 하이라이트해 보세요.
