Marks: Text Highlighter 是一款强大且注重隐私的 Firefox 扩展，可让你在任意网页上高亮并管理文本。它旨在提升效率、优化阅读流程，并让你轻松回看重要内容，同时确保你的数据始终由你掌控。

🔍 主要功能
✅ 多颜色文本高亮
在任意网站选择文本后，你可以用喜欢的颜色进行高亮。扩展提供五种不同的高亮颜色：黄色、绿色、蓝色、粉色和橙色，方便你从视觉上整理和分类内容。

⚡️ 通过右键菜单或快捷键快速高亮
你可以通过以下方式立即高亮选中文本：
- 右键上下文菜单
- 可自定义键盘快捷键（Ctrl+Shift+1 到 Ctrl+Shift+4 等）

这些操作可跨多个标签页和会话使用。

🗂 按页面管理高亮
每条高亮都会与对应 URL 关联并保存。之后重新打开该页面时，所有高亮都会自动恢复。你可以：

- 查看所有已高亮页面列表
- 查看、编辑或删除单条高亮或整组高亮
- 查看页面标题、最后更新时间等元数据

🧭 用于导航的迷你地图视图
页面右侧会显示内置“迷你地图”，并用标记展示所有高亮位置。这能帮助你：

- 在长页面中快速定位高亮内容
- 点击标记直接跳转到对应高亮位置
- 查看当前屏幕可见的高亮

🌐 多语言支持
支持多种语言，UI 会根据浏览器语言设置自动适配。

🧩 弹出式仪表盘
点击扩展图标会打开一个弹窗，快速访问：

- 当前页面中的高亮文本
- 迷你地图开关
- 删除全部高亮或导出数据按钮
- 跳转到管理全部高亮的完整页面

💾 本地优先存储与隐私保护
默认情况下，包括高亮、颜色和页面元数据在内的所有数据都会通过 storage.local API 存储在浏览器本地。若你启用浏览器 Sync，数据可通过浏览器账号在设备间同步。Marks 不运营自己的后端服务器。

💯 免费且开源
该扩展完全免费，并以 MIT 许可证开源。无付费功能，无使用追踪，无数据收集。

🙌 为什么安装 Marks？
- 即时标记关键信息，提升阅读效率
- 使用颜色整理高亮，让学习更高效
- 保留你的可视化笔记，随时重访有价值页面
- 借助直观迷你地图，在长页面中不再迷失
- 通过本地优先存储和可选浏览器 Sync 自主掌控数据
- 通过快捷键和右键菜单节省时间
- 全部功能永久免费

🔐 信任与隐私
Marks 从设计之初就将隐私放在首位：

- 🚫 除浏览器 Sync 外，不与服务器通信
- 🚫 无分析统计
- 🚫 无需登录或账号

✅ 默认以本地方式运行

此扩展非常适合：

- 研究人员
- 学生
- 经常阅读文章或文档的专业人士
- 任何希望在网页中保留并整理关键信息的人

立即安装 Marks，开始以你的方式高亮网页内容：快速、私密、井井有条。
