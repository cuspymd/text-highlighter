Marks: Text Highlighter は、あらゆるウェブページ上でテキストをハイライトして管理できる、強力でプライバシーを尊重した Firefox 拡張機能です。生産性を高め、読書ワークフローを改善し、重要な内容をスムーズに見返せるように設計されています。データをユーザーがコントロールできることを重視した設計です。

🔍 主な機能
✅ 複数色のテキストハイライト
任意のウェブサイトでテキストを選択し、好みの色でハイライトできます。黄色、緑、青、ピンク、オレンジの 5 色を用意しているため、内容を視覚的に整理・分類できます。

⚡️ コンテキストメニューまたはショートカットでクイックハイライト
選択したテキストは以下の方法ですぐにハイライトできます:
- 右クリックのコンテキストメニュー
- カスタマイズ可能なキーボードショートカット（Ctrl+Shift+1 〜 Ctrl+Shift+4 など）

これらの操作は複数タブやセッションをまたいでも機能します。

🗂 ページ単位のハイライト管理
すべてのハイライトは対応する URL に関連付けて保存されます。後でそのページを開き直すと、ハイライトは自動的に復元されます。次のことが可能です:

- ハイライトしたページの一覧表示
- 個別ハイライトやグループ全体の確認・編集・削除
- ページタイトルや最終更新時刻などのメタデータ確認

🧭 ナビゲーション用ミニマップ表示
ページ右端に内蔵の「ミニマップ」が表示され、すべてのハイライト位置をマーカーで示します。これにより:

- 長いページでもハイライト箇所を即座に特定
- マーカーをクリックして該当箇所へ直接移動
- 現在画面内に表示されているハイライトを把握

🌐 多言語対応
複数の言語に対応。UI はブラウザーの言語設定に応じて自動で切り替わります。

🧩 ポップアップダッシュボード
拡張機能アイコンをクリックすると、以下にすばやくアクセスできるポップアップが開きます:

- 現在のページでハイライトしたテキスト
- ミニマップのオン/オフトグル
- すべてのハイライト削除ボタン、またはデータエクスポートボタン
- 全ハイライトを管理するフルページ管理画面への移動

💾 ローカルファースト保存とプライバシー
デフォルトでは、ハイライト、色、ページのメタデータを含むすべてのデータは storage.local API を使ってブラウザー内にローカル保存されます。ブラウザー Sync を有効にした場合、データはブラウザーアカウントを通じてデバイス間で同期されることがあります。Marks は独自のバックエンドサーバーを運用していません。

💯 無料 & オープンソース
この拡張機能は完全無料で、MIT ライセンスのオープンソースです。プレミアム機能なし。利用追跡なし。データ収集なし。

🙌 Marks をインストールする理由
- 重要な内容をその場でマークして読書効率を向上
- 色分けでハイライトを整理し、よりスマートに学習
- 自分の視覚メモを保ったまま有用なページを再訪
- 直感的なミニマップで長文ページでも位置を見失わない
- ローカルファースト保存と任意のブラウザー Sync でデータを自分で管理
- 高速ショートカットとコンテキストメニューで時間短縮
- すべて永続的に無料

🔐 信頼とプライバシー
Marks はプライバシーを最優先に設計されています:

- 🚫 ブラウザー Sync を除き、サーバー通信なし
- 🚫 分析トラッキングなし
- 🚫 ログイン/アカウント不要

✅ デフォルトでローカル動作

この拡張機能は次のような方に最適です:

- 研究者
- 学生
- 記事やドキュメントを頻繁に読むプロフェッショナル
- ウェブ上の重要情報を保持・整理したいすべての人

今すぐ Marks をインストールして、あなたのウェブをあなたのやり方でハイライトしましょう。高速・プライベート・美しく整理された体験を。
