Marks: Text Highlighter は、Webで読む・調べる・整理する人のためのハイライト拡張機能です。
重要な一文をすぐにマークし、ページ単位で管理し、必要なときに迷わず見返せます。

主な機能
- ページ内コントロール UI で即ハイライト: テキストを選択すると表示される UI から、ワンクリックで色を適用できます。
- 複数色 + カスタムカラー: 基本の 5 色（黄・緑・青・ピンク・オレンジ）に加えて、カスタムカラーを追加し、自分のルールで分類できます。
- すばやい操作: 右クリックメニューとキーボードショートカットの両方に対応。
- ページ単位で管理: ハイライトは URL ごとに保存され、再訪時に自動復元されます。
- ミニマップで移動: 長いページでもハイライト位置を把握し、目的の箇所へ素早くジャンプできます。

同期とプライバシー
- `storage.local` を使ったローカルファースト保存
- ブラウザー同期を有効にした場合、`storage.sync` によるデバイス間同期をサポート
- 独自バックエンドサーバーなし、分析トラッキングなし、アカウント登録不要

こんな方におすすめ
- 研究者、学生、ドキュメントを日常的に読む人
- Web上の情報を後で使える形で整理したい人

Marks で、日々のWeb読書をより速く、整理しやすく、再利用しやすく。
