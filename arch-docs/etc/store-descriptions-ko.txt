Marks: Text Highlighter는 웹에서 읽고, 조사하고, 정리하는 사람을 위한 하이라이트 확장 프로그램입니다.
중요한 문장을 빠르게 표시하고, 페이지별로 체계적으로 관리하며, 필요할 때 바로 다시 찾을 수 있습니다.

핵심 기능
- 컨트롤 UI로 즉시 하이라이트: 텍스트를 선택하면 페이지 안에 컨트롤 UI가 나타나고, 원하는 색을 눌러 바로 강조할 수 있습니다.
- 다양한 색상 + 커스텀 색상: 기본 5색(노랑, 초록, 파랑, 분홍, 주황)을 제공하며, 원하는 커스텀 색상을 추가해 나만의 분류 체계를 만들 수 있습니다.
- 빠른 실행 방식: 우클릭 컨텍스트 메뉴와 키보드 단축키를 모두 지원해 상황에 맞게 가장 빠른 방식으로 하이라이트할 수 있습니다.
- 페이지 단위 관리: 하이라이트는 URL 기준으로 저장되어, 다시 방문하면 자동으로 복원됩니다. 목록에서 수정/삭제/정리도 간편합니다.
- 미니맵 탐색: 긴 문서에서도 하이라이트 위치를 한눈에 보고 원하는 지점으로 바로 이동할 수 있습니다.

동기화 및 개인정보 보호
- 기본 저장: storage.local 기반 로컬 저장
- 동기화 지원: 브라우저 동기화 사용 시 storage.sync를 통해 기기 간 하이라이트 동기화
- 자체 서버 미운영: 별도 계정 없이 사용 가능, 추적/분석 코드 없음

이런 분께 추천합니다
- 논문, 기술 문서, 아티클을 자주 읽는 사용자
- 색상으로 핵심 내용을 분류하며 읽고 싶은 사용자
- 여러 기기에서 같은 읽기 맥락을 유지하고 싶은 사용자
