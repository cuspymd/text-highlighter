Simple Installation Process

Install Marks in just 3 steps and start using it right away.

Step 1

Search for "Marks: Text Highlighter" in the Chrome Web Store.

Step 2

Click "Add to Chrome" to install the extension.

Step 3

Select text on any webpage and start highlighting!
