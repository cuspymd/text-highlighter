스마트 컨트롤

텍스트를 선택하기만 하면 직관적인 플로팅 UI가 나타납니다. 추가 클릭 없이 즉시 하이라이트하세요.

🎨

사용자 정의 색상

무제한 사용자 정의 색상으로 개성을 표현하세요. 고급 컬러 피커를 통해 완벽한 색조를 찾을 수 있습니다.

☁️

클라우드 동기화

당신의 하이라이트는 어디든 따라갑니다. 모든 기기와 브라우저 간에 자동으로 동기화됩니다.

🗺️

미니맵

화면 가장자리의 하이라이트 미니맵을 사용하여 긴 기사도 쉽게 탐색할 수 있습니다.

📋

대시보드

여러 웹사이트에 흩어진 하이라이트를 하나의 아름다운 대시보드에서 중앙 집중식으로 관리하세요.

📱

모바일 준비

Firefox Android 등 모바일 브라우저에 최적화되었습니다. 이동 중에도 하이라이트하세요.

직관적인 워크플로우

최신 컨트롤 UI

더 이상 메뉴를 찾아 헤맬 필요가 없습니다. 웹 페이지에서 텍스트를 선택하면 커서 근처에 세련된 하이라이트 아이콘이 나타납니다.

아이콘을 클릭하여 선호하는 색상을 선택하거나, 기존 하이라이트를 탭하여 즉시 색상을 변경하고 삭제할 수 있습니다.

이것은 하이라이트된 문장입니다...

나만의 색상, 나만의 방식

기본 제공 색상에 안주하지 마세요. 고급 컬러 피커를 통해 어떤 색상이든 팔레트에 추가할 수 있습니다.

색상, 채도, 명도를 조정하여 학습 노트나 연구 프로젝트에 완벽한 테마를 만드세요. 확장 프로그램 팝업에서 커스텀 색상을 쉽게 관리할 수 있습니다.

심리스한 클라우드 동기화

데이터 유실 걱정은 이제 그만하세요. Text Highlighter는 모든 기기에서 데이터를 자동으로 동기화합니다.

직장 노트북에서 하이라이트하고 집의 컴퓨터나 모바일 기기에서 바로 확인하세요. Chrome과 Firefox 간의 교차 브라우저 동기화도 지원합니다.
