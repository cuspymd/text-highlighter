웹페이지 텍스트를
스마트하게 하이라이트하세요

Marks는 웹페이지의 중요한 텍스트를 색상별로 하이라이트하고 효율적으로 관리할 수 있는 Chrome 확장 프로그램입니다.

지원하는 기능들

🎨

다중 색상 하이라이트

5가지 색상(노란색, 초록색, 파란색, 분홍색, 주황색)으로 텍스트를 분류하여 하이라이트할 수 있습니다.

⚡

키보드 단축키

Ctrl+Shift+숫자 키를 사용하여 빠르게 하이라이트를 적용할 수 있습니다.

🗺️

미니맵 기능

페이지 우측에 미니맵을 표시하여 하이라이트된 위치를 한눈에 파악할 수 있습니다.

💾

자동 저장

하이라이트한 내용이 자동으로 저장되어 다음 방문 시에도 그대로 유지됩니다.

🌍

다국어 지원

다양한 언어를 지원하여 더 많은 사용자가 편리하게 이용할 수 있습니다.

✨

쉬운 관리

우클릭 컨텍스트 메뉴와 하이라이트 컨트롤러로 쉽게 하이라이트를 편집하고 삭제할 수 있습니다.

실제 사용 예시

텍스트를 선택하고 우클릭하거나 키보드 단축키를 사용하여 원하는 색상으로 하이라이트할 수 있습니다.

  • 텍스트 선택 후 우클릭으로 색상 선택
  • Ctrl+Shift+1~4 키보드 단축키 지원
  • 하이라이트 클릭으로 색상 변경 및 삭제
  • 페이지별 하이라이트 자동 저장
  • 미니맵으로 하이라이트 위치 확인

이것은 중요한 정보입니다. 웹페이지에서 핵심 내용을 쉽게 구분할 수 있습니다.

연구 자료나 학습 내용을 정리할 때 매우 유용하며, 나중에 다시 방문해도 하이라이트가 그대로 유지됩니다.

간단한 설치 과정

3단계만으로 Marks를 설치하고 바로 사용할 수 있습니다.

Chrome 웹스토어에서 설치

1단계

Chrome 웹스토어에서 "Marks: Text Highlighter"를 검색하세요.

2단계

"Chrome에 추가" 버튼을 클릭하여 확장 프로그램을 설치하세요.

3단계

웹페이지에서 텍스트를 선택하고 바로 하이라이트를 시작하세요!
