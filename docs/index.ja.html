ウェブページのテキストをスマートにハイライト

Marksは、ウェブページの重要なテキストを色分けして効率的に管理できるChrome・Firefox拡張機能です。

サポート機能

🎨

複数色のハイライト

5色（黄色、緑、青、ピンク、オレンジ）の基本色とカスタム色でテキストを分類してハイライトできます。

⚡

キーボードショートカット

Ctrl+Shift+数字キーで素早くハイライトを適用できます。

🗺️

ミニマップ機能

ページ右側にミニマップを表示し、ハイライト位置を一目で確認できます。

💾

自動保存

ハイライトした内容は自動で保存され、次回訪問時もそのまま維持されます。

🌍

多言語対応

様々な言語をサポートし、より多くのユーザーが便利に利用できます。

✨

簡単管理

右クリックコンテキストメニューとハイライトコントローラーで簡単に編集・削除できます。

実際の使用例

テキストを選択して右クリックするか、キーボードショートカットを使用して希望の色でハイライトできます。

  • テキスト選択後、右クリックで色を選択
  • テキスト選択後に表示されるアイコンをクリックしてハイライト適用
  • Ctrl+Shift+1〜4のキーボードショートカット対応
  • ハイライトをクリックして色変更/追加や削除
  • ページごとにハイライトを自動保存
  • ミニマップでハイライト位置を確認

これは重要な情報です。ウェブページで核心内容を簡単に区別できます。

研究資料や学習内容を整理するのに非常に便利で、後で再訪問してもハイライトはそのまま維持されます。

簡単なインストール手順

3ステップでMarksをインストールしてすぐに使用できます。

Chromeウェブストアからインストール Firefoxアドオンからインストール

ステップ1

ChromeウェブストアまたはFirefoxアドオンで「Marks: Text Highlighter」を検索してください。

ステップ2

「Chromeに追加」または「Firefoxに追加」ボタンをクリックして拡張機能をインストールしてください。

ステップ3

ウェブページでテキストを選択してすぐにハイライトを始めましょう！
