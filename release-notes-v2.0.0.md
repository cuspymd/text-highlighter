## Text Highlighter v2.0.0

### 주요 변경 사항
- 팝업과 페이지 목록 UI를 전반적으로 개편하고, 모달 컴포넌트를 통합해 일관성을 높였습니다.
- 팝업/페이지 목록의 i18n 제어 및 로컬라이즈드 모달 동작을 정비했습니다.
- 페이지 목록에서 favicon 조회 시 URL 노출 가능성을 줄이도록 보안 처리를 개선했습니다.
- 모바일 환경에서 미니맵 터치 상호작용과 확장 상태 스타일을 개선했습니다.
- 미니맵 툴팁 중복 표시 문제를 수정하고 hover 스타일 동작을 안정화했습니다.

### 동기화/데이터 안정성
- 하이라이트 import 시 `updatedAt` 보존 로직을 보강했습니다.
- bulk delete 동기화 tombstone 회귀 이슈를 수정했습니다.
- 하이라이트 삭제 시 `lastUpdated` 갱신 로직을 보완했습니다.
- 레거시 하이라이트의 누락 필드 처리 및 삭제 전파 경합 처리를 개선했습니다.

### 구조/품질 개선
- background 로직을 책임 단위 모듈로 분리하고 content script 구조를 정리했습니다.
- 배포 스크립트 경로/복사 대상을 정비해 패키징 안정성을 높였습니다.
- Jest/Playwright 테스트를 ESM 환경에 맞게 정비하고 동기화 시나리오 E2E 안정성을 강화했습니다.
